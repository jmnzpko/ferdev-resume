# Fernando Jimenez — Developer Portfolio & Interactive Resume

Welcome to my personal portfolio and interactive resume!  
This repository hosts the source code for my online resume built with **React**, designed to showcase my professional experience, technical skills, and the projects I’ve worked on throughout my career.

This project is based on a customizable React resume template, which I’ve adapted to highlight my work, style, and accomplishments in a clean and modern format.

---

## 🚀 About This Project

This website serves as:

- **My interactive resume** — a digital version of my background, skills, and experience.
- **A project portfolio** — showcasing the applications, tools, and experiments I’ve built over the years.
- **A living document** — continuously updated as I grow as a developer.

By hosting my resume and portfolio as code, I can easily update content, track changes, and demonstrate my familiarity with modern development tools and workflows.
